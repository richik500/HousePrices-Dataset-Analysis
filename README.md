# House Price Data Analysis in R
This repository contains a complete statistical analysis project on house price data using R and RStudio. The project explores pricing trends, customer insights, promoter strategies, and housing characteristics through data visualization and hypothesis testing.

## Project Objective
To analyze house prices based on multiple factors such as:
- Neighborhood
- Square Feet (SqFt)
- Bedrooms
- Bathrooms
- Brick Construction
- Offers Received
The project provides insights from both **Buyer** and **Promoter** perspectives.

## Dataset Features
The dataset includes variables such as:
- Price
- SqFt
- Neighborhood
- Bedrooms
- Bathrooms
- Offers
- Brick

## Key Business Insights
- West neighborhood has the highest average house prices.
- North neighborhood contains the lowest priced houses.
- Brick houses are priced significantly higher than non-brick houses.
- Price has a positive correlation with SqFt, Bedrooms, and Bathrooms.
- Higher priced houses generally receive fewer offers.
- West region shows premium price per SqFt.
